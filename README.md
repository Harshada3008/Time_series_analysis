# 📊 Time Series Analysis Dashboard (Streamlit)

## 🚀 Overview

This project is an interactive **Time Series Analysis Dashboard** built using Streamlit.
It allows users to upload noisy datasets, visualize original data, and apply **Moving Average smoothing** to reduce noise and identify trends.

---

## 🧠 Features

* 📁 Upload CSV file (noisy data)
* 📄 View original dataset
* 📊 Interactive graph visualization
* 🔄 Noise reduction using Moving Average
* 🎚️ Adjustable smoothing window
* ⚡ Real-time updates

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib

## ▶️ How to Run

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/time-series-streamlit.git
cd time-series-streamlit
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run App

```
streamlit run app.py
```
